LinkedMarkMail

LinkedMarkMail is a simple Linked Data interface for accessing the MarkMail archives. Using its hacky API, the SWAML project offers a RDF version of the information indexed by MarkMail; using SIOC, therefore fully compatible with many other applications.

The alpha services is available at: http://linkedmarkmail.wikier.org/, where you can check both messages and threads by ID (for instance this message or this thread). Futher details at its voID description.

http://linkedmarkmail.wikier.org/ /

The service is provided without any warranty, due the source of information actually is not a public API, but a internal API used by MarkMail for improving the user experience with JavaScript. Therefore we may need to close the service at any time if we are required.

But remember that LinkedMarkMail is free software; so if you'd need it, you can check out its source code:

svn checkout http://svn.berlios.de/svnroot/repos/swaml/trunk/apps/prototypes/linkedmarkmail linkedmarkmail
