
\chapter{Ontología}

Se ha desarrollado, teniendo muy en cuenta las buenas prácticas que recomienda el W3C para publicar vocabularios
RDF\footnote{http://www.w3.org/TR/swbp-vocab-pub/}, una sencilla ontología que formalizara las representación 
de una lista de correo que SWAML publica en RDF. 

La URI para la ontología es \url{http://swaml.berlios.de/ns/swaml.owl}, siendo además esa URI la URL donde se 
encuentra la ontología en OWL DL desarrollada.
