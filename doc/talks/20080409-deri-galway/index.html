Mining Online Communities

We considered two approaches:

  • Intrusive techniques
    • SWAML
    • others exporters (WordPress, Drupal, etc.)
  • Unintrusive techniques
    • HTML scraping with XSLT
      • using tidy to clean the markup
      • different outputs:
        • generate RDF/XML
        • generate XHTML+RDFa
    • RSS/Atom translation
